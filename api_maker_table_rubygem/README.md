## Installation
Add this line to your application's Gemfile:

```ruby
gem "api_maker_table"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install api_maker_table
```

Install migrations:
```bash
rails api_maker_table:install:migrations
```

Run migrations:
```bash
rails db:migrate
```

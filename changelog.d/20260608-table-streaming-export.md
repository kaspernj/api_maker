Reworked the table export (Settings → Download) to stream the whole result set to a file page-by-page instead of building one big HTML string in memory from the loaded rows only. Adds a format picker — CSV and HTML stream with bounded memory; Excel is available when the host registers `apiMakerConfig.setExportXlsxSerializer(...)`. On web it streams to disk via the File System Access API (falling back to file-saver), and on React Native/Expo the host provides a sink via `apiMakerConfig.setExportFileSinkOpener(...)`.
